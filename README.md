# 재능대학교 컴퓨터소프트웨어학과 졸업작품 아카이브

재능대학교 컴퓨터소프트웨어학과 학생들의 졸업작품을 모아둔 아카이브 프로젝트입니다. 학생들의 열정과 기술이 담긴 프로젝트들을 영구히 기록하고 공유하기 위한 플랫폼입니다.

## 📋 프로젝트 개요

- **프로젝트명**: JEIU Creative Collection
- **설명**: 졸업작품 아카이브 웹사이트
- **기술 스택**: HTML, CSS, JavaScript
- **기능**: 
  - 팀별 졸업작품 정보 표시
  - 팀원 사진 및 정보 표시
  - 작품 설명 및 플랫폼 정보
  - 중간 및 최종 발표 영상 링크
  - 반응형 이미지 캐러셀
  - 연도별 아카이브 기능

## 📊 데이터 구조

### D1 Database (Teams)
팀 데이터는 이제 D1 데이터베이스에 저장됩니다:

**teams 테이블:**
- `id`: 고유 팀 ID (예: "team_a_01")
- `team_name`: 팀 이름
- `project_name`: 프로젝트 이름
- `members`: 팀원 배열 (JSON)
- `class_name`: 반 이름 ("a반", "b반")
- 기타 프로젝트 정보 필드

### API 엔드포인트
- `GET /api/teams`: 모든 팀 데이터 조회
- `POST /api/teams`: 새 팀 생성

### students.json (레거시)
- `a반`, `b반`: 반별 팀 정보
- `조이름`: 팀명
- `조원`: 팀 구성원 배열
- `작품설명`: 프로젝트 설명
- `예상플랫폼`: 서비스 플랫폼 (데스크톱, 모바일 등)
- `최종발표영상`: 최종 발표 영상 URL
- `중간발표영상`: 중간 발표 영상 URL
- `최종발표자`: 발표자 이름
- `판넬제출여부`: 판넬 제출 여부 ('Y' 또는 공란)

### students.json
- 학생 이름을 키로 하는 객체
- `name`: 학생 이름
- `img`: 학생 사진 경로 또는 placeholder URL

## 🖥️ 기술적인 특징

- **반응형 웹 디자인**: 모바일, 태블릿, 데스크톱에서 최적화된 표시
- **WebGL 배경**: WebGL을 사용한 동적인 배경 효과
- **이미지 캐러셀**: egjs-flicking 라이브러리를 사용한 팀원 이미지 캐러셀
- **비디오 모달**: 발표 영상을 모달 창에서 재생
- **연도별 필터링**: 연도별 아카이브 기능
- **노이즈 효과**: CSS를 사용한 이미지 노이즈 오버레이

## 🚀 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열면 메인 페이지가 표시됩니다.
2. 상단에서 연도를 선택하여 해당 학년도의 졸업작품을 볼 수 있습니다.
3. 각 카드를 클릭하면 팀 정보, 팀원 사진, 작품 설명, 발표 영상을 확인할 수 있습니다.
4. 발표 영상 버튼을 클릭하면 YouTube 영상이 모달 창에서 재생됩니다.

## 🔧 커스터마이징

### 데이터 추가
새로운 졸업작품을 추가하려면 D1 데이터베이스에 직접 데이터를 추가하거나 API를 사용하세요:

**API를 통한 추가:**
```bash
curl -X POST https://jeiu.cc/api/teams \
  -H "Content-Type: application/json" \
  -d '{
    "id": "team_a_06",
    "team_name": "새팀",
    "project_name": "새프로젝트",
    "class_name": "a반",
    "members": ["팀원1", "팀원2"]
  }'
```

**데이터베이스 직접 추가:**
```sql
INSERT INTO teams (
    id, team_name, project_name, class_name, members
) VALUES (
    'team_a_06', '새팀', '새프로젝트', 'a반', '["팀원1", "팀원2"]'
);
```

### 이미지 추가
1. `faces/` 폴더에 학생 얼굴 사진을 추가합니다.
2. 파일 이름은 학생 이름과 일치시킵니다 (예: `홍길동.jpg`).
3. 지원되는 형식: `.jpg`, `.jpeg`, `.png`, `.webp`
